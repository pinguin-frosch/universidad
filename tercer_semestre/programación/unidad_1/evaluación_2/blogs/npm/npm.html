Npm

Npm (node proyect manager) es un gestor de paquetes para node.js, similar a pip para python o nuget para .net. Cuenta con muchas características que solo convierten en una buenísima opción para manejar las dependencias de un proyecto. Al comenzar cualquier proyecto deberíamos iniciar un proyecto de npm y eso automáticamente se encargará de almacenar las versiones de los paquetes que usemos a medida que las actualizemos o las borremos del proyecto. Cualquier persona puede subir paquetes para que luego otras personas los usen a través de npm, es lo bueno de la programación, siempre podemos encontrar personas que comparten código de forma libre y que los demás puedan usar.
