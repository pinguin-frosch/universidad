Resumen de carrera

Mi carrera se podría decir que empieza hace unos años atrás. El año 2020 fue cuando comenzé a hacer cosas relacionadas con la programación. Lo primero que realicé fueron unos proyectos pequeños relacionados al desarrollo web, pero me dí cuenta que no era lo que más me gustaba. Un tiempo después cuando volví a la programación y tomé python, y desde ese entonces amé la programación. Al día de hoy sigo aprendiendo diferentes cosas relacionadas al tema.

Proyectos en que he trabajado

Inacapi

Proyecto personal
2021 - Presente

Inacapi es un proyecto que como el nombre sugiere, es una api para inacap. La razón de este proyecto es automatización, me resulta más cómodo tener acceso a los datos de inacap en una hoja de cálculo de Google, de esa forma la puedo ver de cualquier parte. Además programé funciones adicionales que calculan el promedio, los días restantes para una evaluación, etc. El programa se ejecuta en segundo plano así que ni siquiera le tengo que dar mucha atención. En lo que estoy trabajando ahora es en añadir el resto de la información que proporciona inacap y aún no obtengo, como los compañeros, la asistencia, etc.

Logros:

  • Familiarización con nodejs y npm para gestionar paquetes.
  • Aprender la estructura de proyectos en nodejs.
  • Uso de funciones externas en las hojas de cálculo de google.

Tecnologías usadas:

  • Javascript
  • Nodejs
  • NPM
  • Api de google sheets

Instalador automático de Arch Linux

Proyecto personal
2021

La finalidad de este proyecto es poder instalar Arch Linux de forma automática, porque puede ser muy tedioso hacerlo de otra forma. Es automático, pero no es tan personalizable, la mayoría de las configuraciones son parte del código y no de un archivo aparte, sin contar los programas a instalar. No recomiendo que alguien lo uso, hay mejores alternativas que son mucho mejores, pero yo prefiero usarlo, porque se adapta perfectamente a lo que necesito.

Logros:

Este proyecto me ayudó a comprender cómo funcionan los scripts en bash, que son muy comunes en Linux.

Tecnologías usadas:

  • Bash
  • Arch Linux

Color js

Proyecto personal
2022

Color js es una extensión que hice hace un tiempo, permite cambiar el fondo de un sitio web. El color que se usará viene incluido en la página de opciones, hay 12 colores por defecto, pero todos ellos se pueden modificar a gusto y son guardados en la memoria del navegador o en la cuenta de google si se tiene activada la sincronización.

Tecnologías usadas:

  • JavaScript
  • NPM
  • Chrome api
