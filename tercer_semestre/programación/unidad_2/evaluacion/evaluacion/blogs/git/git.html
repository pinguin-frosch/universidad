Git

Git es un programa que permite controlar diferentes versiones de un archivo. Esto es muy útil en programación porque si uno hace cambios que no funcionan se pueden revertir fácilmente y también permiten una colaboración sencilla con equipos de personas. Existen alternativas a git pero no son muy populares, muchas personas ni siquiera las conocen, algunos ejemplos son cvs, svn, mercurial y bazaar. Todo el historial que maneja git es local, pero permite conectarse a aplicaciones que permitan almacenar los proyectos en la nube, como github, pero hay más opciones igualmente, como gitlab o bitbucket.
